SMILE Lab RampUp
===================================

This is for new trainees joining the SMILE Lab. These pages and accompanying exercises are designed to get one familiar with the data structures that are used in holding behavioral and neural data collected in the lab as well as some of the common processing and analysis techniques that are commonly employed to answer scientific questions from it.


.. note::

   This project is under active development.
